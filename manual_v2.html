프로그램 소개

이레캔 애드온이란?

게임 플레이를 편리하게 도와주는 매크로 프로그램입니다.

💡 편리한 ON/OFF 기능
단축키로 이레캔 애드온 기능을 쉽게 ON / OFF 할 수 있습니다.
게임창이 비활성화 되면 자동으로 OFF 됩니다.
OFF 상태에서는 원래 키로 동작합니다.

ON/OFF 토글 버튼 - 클릭하거나 핫키로 전환

주요 기능

  • 핫키 매핑 F5~F12 키를 자주 쓰는 키 Q W E R 등으로 변경
  • 휠 클릭 단축키를 마우스 휠 버튼 클릭으로 사용할 수 있음
  • 시퀀스 실행 여러 동작을 순서대로 자동 실행
  • 오버레이 버튼 애드온 화면에 버튼을 띄워서 클릭 가능

첫 실행 시작하기

STEP 1. 프로그램 실행

  1. IreCan Smart Addon.exe 파일을 더블클릭하여 실행합니다.
  2. 관리자 권한 승인 창이 뜨면 예를 클릭합니다.
⚠️ 중요: 관리자 권한이 없으면 프로그램이 작동하지 않습니다.

STEP 2. 라이선스 인증

📌 신규 사용자

프로그램을 처음 실행하면 라이선스 등록 창이 나타납니다.

단계별 진행:

  1. 사용자명 입력 (예: 홍길동)
    • 영문, 한글, 숫자만 가능
    • 특수문자 불가
  2. 연락처 입력 (예: 01012345678 또는 @telegram_id)
  3. 등록 신청 버튼 클릭
  4. 텔레그램 버튼을 눌러 관리자에게 문의
  5. 관리자 승인 후 프로그램을 다시 실행

📌 기존 사용자 (라이선스 만료)

사용 기간이 만료되면 연장 신청 창이 나타납니다.

  1. 사용자명/연락처 확인 (변경 가능)
  2. 연장 신청 클릭
  3. 텔레그램으로 관리자 문의
  4. 승인 후 프로그램 재실행

STEP 3. 게임창 지정 (필수!)

라이선스 인증이 완료되면 게임창 지정 창이 나타납니다.

  1. 확인 버튼을 클릭합니다.
  2. 게임 창 위에 마우스 커서를 올리고 한 번 클릭합니다.

게임창 지정 - 리니지 클래식 창을 클릭하세요

⚠️ 데스크톱이나 다른 프로그램 창을 클릭하면 안 됩니다!
✅ 선택 완료 메시지: 게임창 지정 완료: Lineage Classic

기본 설정 (필수)

게임창을 선택했다면, 이제 핫키 매핑과 좌표를 설정해야 합니다.

트레이 아이콘 메뉴

화면 오른쪽 하단 시스템 트레이에서 IreCan Smart Addon 아이콘을 우클릭하면 메뉴가 나타납니다.

또는, 화면에 나타난 ON / OFF 버튼에 마우스 우클릭 해도 메뉴가 나타납니다.

트레이 아이콘 우클릭 메뉴 - 모든 설정에 접근 가능

STEP 4. 단축창 좌표설정

목적: F5~F12 단축키의 화면 위치를 먼저 지정

  1. ON / OFF 버튼 (또는 트레이 아이콘) 우클릭 → [필수]단축창 좌표설정 클릭
  2. F5 좌표 버튼 클릭
  3. 안내 메시지: 단축창 위치에서 우클릭 하세요.(5초 타임아웃)
  4. 게임 창에서 해당 단축키 위치를 마우스 우클릭
  5. 자동으로 좌표가 저장됨: F5: (645, 873)
  6. F6~F12도 같은 방법으로 설정
  7. 확인 클릭하여 저장

단축창 좌표설정 - 각 단축키 위치를 우클릭으로 지정

STEP 5. 단축창 핫키설정

목적: F5~F12 키를 자주 쓰는 키(QWER)로 변경

  1. 트레이 아이콘 우클릭 → 단축창 핫키설정 클릭
  2. ON / OFF 키 변경 (선택사항)
    • ` 버튼 클릭 후 원하는 키를 누름
    • ESC를 누르면 취소
  3. F5~F12 키 매핑 변경
    • 예: F5를 Z로 바꾸려면 Q 버튼 클릭 후 Z 키 입력
  4. 휠 클릭 설정
    • F5~F12만 휠 클릭 가능
    • 체크박스 체크 설정하고 Q 키를 누르면 단축창 F5 + 휠 버튼 클릭으로 동작
  5. 프리셋 전환 (A, B, C)
    • 예: A = 기사, B = 요정, C = 법사
  6. 확인 클릭하여 저장

단축창 핫키설정 - F5~F12를 원하는 키로 매핑

타이밍 설정 (고급)

목적: 클릭 속도 조절 (기본값으로도 충분함)

💡 추천 설정: 기본값 그대로 사용 (건드리지 않아도 됨)

타이밍 설정 - 클릭 속도 미세 조정 (고급 사용자용)

기본 기능 사용하기

STEP 6. 매크로 ON/OFF

방법 1: 키 매핑 설정에서 지정한 키 사용

  • 기본값: ` 키 (숫자1 왼쪽에 있는 키)
  • 게임 창이 활성화된 상태에서 ` 누름
  • OFF에서 ON으로 전환

방법 2: OFF / ON 버튼 직접 마우스 클릭

  • 좌클릭으로 ON / OFF 전환

STEP 7. 매핑된 핫키 사용하기

예시: F5 키를 Q로 매핑, 휠클릭 체크

  1. 매크로 ON 상태로 전환
  2. 게임 창에서 Q 키를 누름
  3. 프로그램이 자동으로:
    • F5에 설정한 좌표 (645, 873) 인식
    • 휠 버튼 클릭 (휠클릭이 ON인 경우)
    • 마우스 포인터 위치 변동 없음
✅ 결과: 현재 마우스 위치에 바로 F5 휠클릭

핫키 반복 기능

매크로가 ON 상태에서 Shift + 핫키(QWER) 입력: 해당 단축키 무한반복

반복 해제는 해당 단축키만 다시 입력 (Shift 안 누름!!)

💡 툴팁 숨기기: 트레이 아이콘 우클릭 → "툴팁 보이기/숨기기"

게임창 비활성화 시 자동 OFF

안전 기능:

  • 게임 창에서 다른 창으로 전환하면 자동으로 매크로 OFF
  • 다른 프로그램에서 핫키가 작동하는 것을 방지, 바로 기본 키보드로 사용

고급 기능

버튼 UI

목적: 화면에 버튼을 띄워서 마우스로 클릭 가능

버튼 UI 표시:

  1. 트레이 아이콘 우클릭 → 버튼 보이기/숨기기

버튼 UI - 화면에 오버레이로 표시되는 클릭 가능한 버튼들

버튼 클릭:

  • 버튼을 마우스로 클릭하면 지정된 시퀀스가 실행

버튼 UI 설정:

  1. 버튼 아무데나 Ctrl + 우클릭 드래그 → 원하는 곳으로 이동
  2. ON / OFF 버튼 또는 트레이아이콘에 우클릭 - 버튼UI 설정 메뉴 선택
    • 버튼 개수: 1~20개 필요한 개수 설정
    • 행 수: 1~3줄 선택
    • 방향: 가로/세로 선택
    • 버튼 너비: 20~200 범위에서 설정
    • 버튼 높이: 20~200 범위에서 설정
    • 버튼 간격: 0~20 범위에서 설정
    • 투명도: 0~255 슬라이더로 설정

버튼 UI 설정 - 개수, 크기, 배치 등 커스터마이징

버튼 우클릭 메뉴 - 핫키 지정 및 시퀀스 설정

버튼 반복 실행

목적: 버튼을 한 번 클릭하면 계속 반복 실행

  1. 버튼 UI에서 원하는 버튼 Shift + 좌클릭
    • 해당 버튼에 지정된 시퀀스가 무한 반복 실행
    • 실행 중인 버튼에 텍스트가 청록색으로 깜빡임
  2. Shift + 핫키(QWER ASDF 등)
    • 해당 핫키가 무한 반복 실행
  3. 반복 해제
    • 해당 버튼 좌클릭
    • 해당 핫키 입력
    • 매크로 OFF

시퀀스 설정

목적: 여러 동작을 녹화하고 자동 재생

시퀀스 설정 열기

  1. UI버튼 / 트레이 아이콘 우클릭 → 시퀀스 설정

시퀀스 설정 - 여러 동작을 순서대로 녹화하여 자동 실행

시퀀스 만들기

예시: "데스변신" 시퀀스

  1. 새 시퀀스 클릭
  2. 시퀀스 이름 입력: ex 데스변신
  3. 확인 버튼 클릭

시퀀스 핫키 지정

  1. 버튼 UI에서 버튼 우클릭
  2. 핫키 지정 선택
  3. 드롭다운에서 시퀀스 선택
  4. 지정한 핫키로 시퀀스 실행

버튼에 시퀀스 지정 - 버튼 클릭으로 시퀀스 실행

💡 참고: 시퀀스 상세 가이드는 추후 추가 예정입니다.

문제 해결

❓ 프로그램이 실행되지 않아요

해결 방법:

  1. 관리자 권한으로 실행했는지 확인
    • UAC 승인 창에서 "예" 클릭 필수
  2. 라이선스 등록 및 승인 확인
    • 안내 메시지 확인

❓ 키를 눌러도 반응이 없어요

체크리스트:

  • 매크로가 ON 상태인가요? 초록색 ON 버튼 확인
  • 게임 창이 활성화되어 있나요?
  • 단축창 좌표설정을 했나요?
  • 단축창 핫키설정을 했나요?

해결 방법:

  • OFF 버튼이나, 트레이 아이콘 우클릭 → [필수]게임창 지정으로 다시 선택
  • 좌표/핫키 재설정
  • 게임창에서 OFF 버튼 클릭 → ON으로 전환

❓ 다른 창에서도 키가 눌려요

원인: 게임창 지정이 잘못됨

해결 방법:

  1. ON / OFF 버튼이나, 트레이 아이콘 우클릭 → [필수]게임창 지정
  2. 정확히 게임 창 위를 클릭

❓ 휠 클릭이 안 눌려요

체크리스트:

  • F5~F12 키만 휠 클릭 가능 (F1~F3은 불가)
  • 키 매핑 설정에서 휠클릭 체크박스 체크 확인

해결 방법:

  • ON / OFF 버튼이나, 트레이 아이콘 우클릭 → 단축창 핫키설정
  • F5~F12 옆 체크박스 확인

❓ 버튼 UI가 안 보여요

해결 방법:

  1. 실행 중인지 확인: 트레이아이콘에 IreCan Smart Addon 확인
  2. 트레이 아이콘 우클릭 → "버튼 보이기/숨기기"

❓ 라이선스가 만료되었어요

해결 방법:

  1. 프로그램 실행 시 연장 신청 창 표시
  2. 연장 신청 클릭
  3. 텔레그램 버튼으로 관리자 문의
  4. 관리자 승인 후 프로그램 재실행

❓ 시퀀스가 실행되지 않아요

체크리스트:

  • 매크로가 ON 상태인가요? 매크로 OFF 상태에서는 버튼 클릭만 작동합니다.
  • 게임 창이 활성화되어 있나요?
  • 시퀀스에 액션이 추가되어 있나요?
  • 이미 실행 중인 시퀀스가 있나요?
  • 시퀀스를 버튼에 지정했나요?

해결 방법:

  • 빨간색 OFF 버튼 클릭해보세요. 초록색 ON으로 바뀌는지 확인
  • 게임창 활성화 상태에서 시도
  • 버튼에 텍스트 색깔이 흰색인지 확인
  • 시퀀스 설정에서 액션 확인
  • 버튼 우클릭 - 시퀀스 지정
