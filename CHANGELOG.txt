[v25.11.14] 2025-11-14
----------------------------------------------
🆕 새로운 기능
- 액션 녹화 중 정지 버튼 추가 (게임 화면 좌상단에 표시)
- 딜레이 최적화 기능 추가 (긴 딜레이 일괄 단축)
- 매크로 자동 복원 기능 추가
  · 다른 창 클릭 시: 매크로 자동 OFF
  · 게임 창 복귀 시: 이전 ON 상태였다면 자동 ON
- 시퀀스 실행 테스트 완료 후 설정 창 자동 활성화
  · 정상 완료 시 설정 창으로 자동 복귀
  · End 키 중단 시에도 설정 창으로 자동 복귀
  · 실행 정지 버튼 클릭 시에도 설정 창으로 자동 복귀

🔧 개선 사항
- 액션 녹화 시 연속 휠스크롤 자동 통합 (1개 액션으로)
- 시퀀스 설정 창 버튼 레이아웃 개선 (↑↓ 버튼 크기 최적화)
- 녹화 정지 버튼 클릭 시 시퀀스 창 자동 활성화
- 게임 창 전환 시 사용자 편의성 대폭 향상

🐛 버그 수정
- 시퀀스 더블클릭 작동 안 되던 문제 해결
